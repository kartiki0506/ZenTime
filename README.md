# ZenTime
Zentime is the new generation time management application designed to help individuals plan their daily routines efficiently by generating a personalized timetable on the basis of urgency, importance and due dates given by the user.
# Overview of the Project 
ZenTime is a Smart Time Table generator application built using Python 
Programming. It helps the users to efficiently organize their daily 
schedule according to their tasks deadlines, commitments and person 
care. The program allows you to manage your time in an effortless 
manner while keeping your mental and physical health in check. It 
eliminate overlapping of tasks, overcommitment and while managing 
your personal time as well, ensuring that your day has a free slot to 
yourself which you can use for personal purpose or devoting this time 
to family. 
The application has 3 key views – Dashboard, Editor, and Analysis- 
enabling users to visualize their routine. The graph helps user to 
analyze their life routines and change it accordingly. Data is stored 
locally in JSON format, allowing persistent storage of schedules across 
sessions. 
 
# Features 
* Interactive GUI Dashboard: The Dashboard displays the current 
schedule saved by the user, total events, and mood indicators 
(smooth/hectic day).  
* Smart Task Scheduling: The program is designed to automatically 
allocate the task based on their importance, urgency and due date.  
* Editor Mode: User can add task and class or remove tasks and 
classes. They can then save the timetable and generate a pie chart to 
analyze the data. The data can be edited again by the user.  
* Analytics: This page displays a pie chart on the basis of tasks, classes 
and free time slots entered by the user. This helps the user to analyze 
their schedule and change accordingly.  
* Data Storage: Schedules saved in JSON format for reuse and updates. 
 
# Technologies / Tools Used 
* Programming Language: Python  
* Libraries: tkinter, ttk, json, datetime, os, math 
* IDE (recommended): Visual Studio Code / IDLE 
* Data Storage: Local JSON file. 
 
# Steps to Install & Run the Project 
* Install Python or above from https://python.org and ensure it’s 
added to PATH.
* Save the file 'ZenTime.py' in a project folder. 
* Open Command Prompt  
* Run the command: python "ZenTime.py" 
* The app window (Zenith - Smart Scheduler) will launch 
automatically. 
 
# Instructions for Testing 
1. Launch the application by running the Python file. 
2. Navigate to the Editor tab and select a day/date. 
3. Add your required tasks with duration and priority. 
4. You can also Navigate from “+Add Task” to “+Add Class/Fixed”  
5. Add your Classes/Meetings. 
6. Click 'Save & Generate' to create a schedule. 
7. Go to the Dashboard tab to view your timeline and task 
completion. 
8. You can click on the checkbox once your task has been completed 
and On the Editor Tab click on Save & Generate to update the 
Dashboard. 
9. Visit the Analytics tab to see the time distribution pie chart. 
10. Modify any day’s schedule by going to the Editor Tab, click 
on Load/create and by selecting the task on Draft items, you can 
remove it by clicking “Remove Selected”
